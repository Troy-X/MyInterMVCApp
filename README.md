# My ASP.NET MVC Projects Portfolio

This repository contains **five mini ASP.NET MVC projects** I developed to practice C# programming, web development, and software engineering principles. Each project demonstrates CRUD operations, MVC architecture, user interface design with Bootstrap, and basic database interactions using Entity Framework.

---

## Projects Overview

### 1. Student Management System
**Description:**  
A simple system to manage student records including adding, editing, deleting, and viewing students.  
**Features:**  
- Add, edit, delete, and list students  
- Search students by name or class  
- Pagination (if implemented)  
**Technologies:** C#, ASP.NET MVC, Entity Framework, Bootstrap, SQL Server  

---

### 2. Meeting Scheduler
**Description:**  
An app to schedule and manage meetings, with UTC-to-local time conversion based on user timezone.  
**Features:**  
- Create, edit, delete, and view meetings  
- Display meetings in local time  
- Timezone-aware scheduling  
**Technologies:** C#, ASP.NET MVC, Entity Framework, Bootstrap, SQL Server
